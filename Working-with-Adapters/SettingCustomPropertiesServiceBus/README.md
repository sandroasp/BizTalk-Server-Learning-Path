# Setting Custom Properties for Azure Service Bus
The Service Bus (SB-Messaging) adapter is used to receive and send from Service Bus entities like Queues, Topics, and Relays. You can use the SB-Messaging adapter to connect your on-premises BizTalk Server to Azure.

## Introduction
Project that explains How to set Custom Properties when sending message to Service Bus from BizTalk Server.
In this article, we will explore how to:
 * Define custom BizTalk context properties
 * Set them in an orchestration
 * Set them in a custom send pipeline component
 * Ensure they are transmitted as Service Bus message properties
 * Verify everything using Service Bus Explorer
