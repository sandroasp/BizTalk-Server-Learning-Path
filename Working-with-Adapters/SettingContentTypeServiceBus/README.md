# Setting Content-Type for Azure Service Bus
The Service Bus (SB-Messaging) adapter is used to receive and send from Service Bus entities like Queues, Topics, and Relays. You can use the SB-Messaging adapter to connect your on-premises BizTalk Server to Azure.

## Introduction
Project that explains How to set the Content-Type when sending message to Service Bus from BizTalk Server.
The SB-Messaging adapter provides a set of context properties, specific to SB-Messaging, for use in your applications. You can use these properties in filter expressions and in your orchestrations.
Namespace: http://schemas.microsoft.com/BizTalk/2012/Adapter/BrokeredMessage-properties
